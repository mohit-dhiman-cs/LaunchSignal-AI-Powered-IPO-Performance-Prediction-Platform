# LaunchSignal 🚀

AI-powered IPO forecasting and analytics platform that predicts IPO listing performance using Machine Learning, market sentiment, GMP, subscription data, and financial indicators.

## 📌 Overview

LaunchSignal helps investors evaluate upcoming IPOs through data-driven predictions. The platform analyzes historical IPO trends and current market signals to estimate potential listing gains/losses, risk levels, and confidence scores.

## ✨ Features

* Predict IPO listing returns using ML models
* Analyze Grey Market Premium (GMP) and subscription data
* Risk score and confidence score generation
* Interactive analytics dashboard
* Real-time API-based predictions
* Scalable full-stack architecture
